逻辑思维

我喜欢挑战自己，而编程恰好充满挑战性。更重要的是，我的逻辑思维因此得到提高。在过去几年里，我先后学了 JAVA、Android、HTML、 CSS、JavaScript、C 和 SQL 等多种计算机语言，我还在不断学习新的编程语言，比如 python 和 node.js。

我爱探索

我设计各种各样的东西，从手机APP到各式网页，再到各种好玩的小程序。

我很喜欢把我的创造性发挥到极限，更享受从无到有、渐渐完美的开发过程。代码已是生命的一部分，相信它会让生活更美好。

但是，探索的旅途并非一帆风顺，我时常会遇到各种BUG，甚至会心灰意冷，但我怀有一颗真心，我坚信一定会遇见所有的美好。

现在，我正经历生命中最好的年华。怀揣着青春，享受属于这个年纪的时光。

我很乐意与你们分享开心的事情。
